# 🤖 AI Chatbot Project

This is an AI-powered chatbot built using:

- 🧠 Transformers (DialoGPT / Groq API)
- 🐍 Python (Flask / Streamlit)
- ⚛️ React Native (Mobile UI)

---

## 🚀 Features

- Real-time chatbot responses
- ChatGPT-like UI (bubbles, avatars, timestamps)
- Fast response optimization
- Streamlit web app support
